# ASSUMPTION: Key Business Assumptions

**Date**: 2026-04-25
**Type**: ASSUMPTION
**Slug**: key-business-assumptions

## Key Assumptions

1. **Merchant data sharing**: Merchants are willing to share sales or surplus data
2. **Value perception**: Merchants see value in prediction-based decision support
3. **Pattern predictability**: Cafés and bakeries have predictable surplus patterns
4. **Willingness to pay**: Merchants may eventually pay for waste reduction, recovery tracking, or reporting features

## Risks if Assumptions Fail

| Assumption          | Risk if False                             |
| ------------------- | ----------------------------------------- |
| Data sharing        | No model training data; can't personalize |
| Value perception    | No merchant adoption                      |
| Predictability      | Model predictions inaccurate              |
| Payment willingness | No revenue model                          |

## Mitigation

- Start with synthetic data (no real data risk)
- Demonstrate value through dashboard
- Validate assumptions during Phase 2 merchant pilot
